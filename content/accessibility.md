## Commitment

This site aims to meet [WCAG 2.2 Level AA](https://www.w3.org/WAI/WCAG22/quickref/) standards. Accessibility is not an afterthought here — it's built into the foundation.

## What's Built In

- **Dark mode** — respects your operating system preference, or toggle manually
- **Reduced motion** — respects your OS preference, or toggle manually; kills all animations and transitions
- **Dyslexic-friendly font** — switches all text to OpenDyslexic
- **Text size control** — small, medium, or large
- **Wide line spacing** — increases line height and paragraph gaps for easier reading
- **Colorblind-safe mode** — adds shapes, borders, and labels so nothing relies on color alone
- **Skip-to-content link** — keyboard users can bypass navigation
- **Breadcrumb navigation** — always know where you are
- **Print stylesheet** — clean formatting when printed, navigation removed
- **Self-hosted fonts** — no external requests, works behind firewalls and content filters
- **No-JavaScript fallback** — interactive tools show a helpful message when JS is unavailable

All accessibility settings persist across pages and visits.

## Known Limitations

- Interactive tools in the Tools section require JavaScript to function
- Some older browsers may not fully support the CSS custom properties used for theming
- Third-party content linked from this site (YouTube videos, external resources) may not meet the same accessibility standards

## Reporting Barriers

If something on this site doesn't work for you — whether it's a screen reader issue, a keyboard trap, a contrast problem, or anything else — please [open an issue on GitHub](https://github.com/mattiebrandolini/denovo/issues) describing what happened and what you expected. I take these seriously and will work to fix them.

You can also use the "Report an issue" link in the footer of any page.
