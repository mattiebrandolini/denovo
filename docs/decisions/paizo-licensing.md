# Decision: PF2e Content Licensing

**Date:** 2026-02-25
**Status:** Active

## Context
The site publishes homebrew rules for Pathfinder Second Edition. Paizo has specific requirements for community-created content that references their IP.

## Decision
- Original content: CC BY-NC-SA 4.0
- PF2e references: Published under Paizo's Community Use Policy
- Any ORC-licensed mechanics: identified where they appear
- Dedicated `/licenses/` page documents all of this
- Footer links to licenses page on every page

## Requirements
- Must include Paizo's Community Use Policy statement verbatim
- Cannot charge for access to content that references Paizo IP
- Cannot imply Paizo endorsement
- Must link to paizo.com
